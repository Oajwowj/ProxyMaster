## ProxyMaster
ProxyMaster is a tool to fetch free proxies from [free-proxy-list.net](https://free-proxy-list.net)

## Python3
The tool is supporting python3 only

## Installation

Clone the repository:

```bash
git clone https://github.com/Bl4ckB1t/ProxyMaster.git
```
## Running
How To Run The Tool

```bash
cd ProxyMaster 
pip install -r requirements.txt
python3 ProxyMaster.py
```
If the command 'pip install -r requirements.txt' doesn't work, Run these commands

```bash
pip install requests
pip install beautifulsoup4
pip install pandas
pip install pycountry
```
